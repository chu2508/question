# 外包技术测试题

之前参与过的外包项目，做的测试题

## Pre-Requisites

- Node
- Npm

## Installation Instructions
Install:
    ```yarn bootstrap```

This installs everything for the App as well as the Server, so please don't run ```npm install``` (unless you also want to run it from the `server` folder)

## Running the App
Run:
    ```yarn start```

## Testing the App
Run:
    ```yarn test```
